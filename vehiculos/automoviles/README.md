# Automoviles

## Objetivo documental

Documentar automoviles para simulacion: direccion, aceleracion, frenado, transmision, tablero, ayudas de conduccion, normas de transito y comportamiento fisico basico.

## Documentos a crear

- `mandos/manual-mandos-automovil.md`
- `operacion/principios-automovil.md`
- `historia/historia-automovil.md`
- `reglamentos/reglamentos-automovil.md`
- `simulacion/diseno-simulador-automovil.md`

## Mandos principales

- Volante.
- Acelerador.
- Freno.
- Embrague, si corresponde.
- Palanca selectora o de cambios.
- Freno de estacionamiento.
- Luces.
- Limpiaparabrisas.
- Intermitentes.
- Tablero.

## Temas clave

- Traccion delantera, trasera e integral.
- Distancia de frenado.
- Subviraje y sobreviraje.
- Cambios manuales y automaticos.
- Seguridad activa y pasiva.
- Reglamentos de circulacion.
