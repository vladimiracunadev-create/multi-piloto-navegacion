# Portaviones

## Objetivo documental

Documentar portaviones como sistemas navales complejos: navegacion, cubierta de vuelo, coordinacion, seguridad, historia y simulacion de operaciones no sensibles.

## Documentos a crear

- `mandos/manual-mandos-portaviones.md`
- `operacion/principios-portaviones.md`
- `historia/historia-portaviones.md`
- `reglamentos/reglamentos-portaviones.md`
- `simulacion/diseno-simulador-portaviones.md`

## Mandos principales

- Puente de mando.
- Control de propulsion.
- Timon.
- Sistemas de navegacion.
- Comunicaciones.
- Gestion de cubierta simulada.
- Control de danos simulado.

## Temas clave

- Coordinacion de tripulacion.
- Seguridad en cubierta.
- Navegacion de grandes buques.
- Historia de la aviacion naval.
- Simulacion responsable.
- Limites de informacion militar moderna.
