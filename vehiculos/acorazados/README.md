# Acorazados

## Objetivo documental

Documentar acorazados desde una perspectiva historica y de simulacion general: arquitectura naval, propulsion, puente, tripulacion, blindaje, historia y representacion responsable.

## Documentos a crear

- `mandos/manual-mandos-acorazado.md`
- `operacion/principios-acorazado.md`
- `historia/historia-acorazado.md`
- `reglamentos/reglamentos-acorazado.md`
- `simulacion/diseno-simulador-acorazado.md`

## Mandos principales

- Gobierno del buque.
- Control de propulsion.
- Comunicaciones internas.
- Instrumentos de navegacion.
- Control de danos simulado.
- Alarmas.

## Temas clave

- Historia naval.
- Blindaje y compartimentacion a nivel general.
- Roles de tripulacion.
- Maniobra de grandes buques.
- Limites de informacion militar.
- Simulacion historica no tactica.
