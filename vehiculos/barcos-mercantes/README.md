# Barcos mercantes

## Objetivo documental

Documentar barcos mercantes para simulacion de navegacion, maniobra, propulsion, carga, puente de mando, comunicaciones, reglamentos maritimos y seguridad.

## Documentos a crear

- `mandos/manual-mandos-barco-mercante.md`
- `operacion/principios-barco-mercante.md`
- `historia/historia-barco-mercante.md`
- `reglamentos/reglamentos-barco-mercante.md`
- `simulacion/diseno-simulador-barco-mercante.md`

## Mandos principales

- Timon.
- Telegrafo de maquina o control de potencia.
- Controles de propulsores auxiliares.
- Radar.
- GPS o sistemas de posicionamiento.
- Cartas nauticas.
- Radio.
- Alarmas.

## Temas clave

- Inercia de grandes masas.
- Maniobras lentas.
- Navegacion costera y oceanica.
- Reglas de colision.
- Carga y estabilidad.
- Clima y mar.
