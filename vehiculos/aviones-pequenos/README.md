# Aviones pequenos

## Objetivo documental

Documentar aviones pequenos para simulacion educativa: sustentacion, ejes de control, instrumentos basicos, reglas de vuelo, seguridad y procedimientos simplificados.

## Documentos a crear

- `reglamentos/reglamentos-avion-pequeno.md`
- `simulacion/diseno-simulador-avion-pequeno.md`

## Mandos principales

- Yugo o baston.
- Pedales de timon.
- Acelerador.
- Mezcla, si aplica.
- Flaps.
- Compensador.
- Instrumentos de vuelo.
- Radio.

## Temas clave

- Sustentacion.
- Cabeceo, alabeo y guinada.
- Velocidad aerodinamica.
- Altitud.
- Meteorologia basica.
- Reglamentos aeronauticos.
