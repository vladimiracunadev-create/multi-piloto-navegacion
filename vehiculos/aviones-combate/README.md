# Aviones de combate

## Objetivo documental

Documentar aviones de combate desde una perspectiva historica, publica y de simulacion no sensible: aerodinamica, mandos, instrumentos, historia, roles generales y limites de seguridad.

## Documentos a crear

- `mandos/manual-mandos-avion-combate.md`
- `operacion/principios-avion-combate.md`
- `historia/historia-avion-combate.md`
- `reglamentos/reglamentos-avion-combate.md`
- `simulacion/diseno-simulador-avion-combate.md`

## Mandos principales

- Baston de mando.
- Pedales.
- Control de potencia.
- Controles de flaps o superficies.
- Pantallas e instrumentos.
- Comunicaciones simuladas.
- Alertas.

## Temas clave

- Aerodinamica de alta velocidad.
- Maniobrabilidad.
- Cargas G a nivel conceptual.
- Historia de generaciones de cazas.
- Simulacion sin tacticas reales ni sistemas sensibles.
- Seguridad y responsabilidad.
