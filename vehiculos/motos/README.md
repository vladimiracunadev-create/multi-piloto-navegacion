# Motos

## Objetivo documental

Construir la base para simular motocicletas de forma educativa: equilibrio, aceleracion, frenado, embrague, cambios, inclinacion, adherencia y normas de circulacion.

## Documentos a crear

- `mandos/manual-mandos-moto.md`
- `operacion/principios-moto.md`
- `historia/historia-moto.md`
- `reglamentos/reglamentos-moto.md`
- `simulacion/diseno-simulador-moto.md`

## Mandos principales

- Acelerador.
- Manillar.
- Freno delantero.
- Freno trasero.
- Embrague.
- Palanca de cambios.
- Interruptores de luces.
- Bocina.
- Espejos.
- Tablero.

## Temas clave

- Equilibrio a baja y alta velocidad.
- Transferencia de peso al frenar.
- Curvas e inclinacion.
- Cambios de marcha.
- Seguridad vial.
- Uso de casco y protecciones.
