# Submarinos

## Objetivo documental

Documentar submarinos a nivel publico y educativo: flotabilidad, inmersion, propulsion, navegacion, soporte vital, sensores generales, historia y limites de seguridad.

## Mandos principales

- Control de profundidad simulado.
- Control de rumbo.
- Control de velocidad.
- Indicadores de presion y profundidad.
- Comunicaciones simuladas.
- Alarmas.
- Gestion de energia simulada.

## Temas clave

- Flotabilidad.
- Lastre.

- Navegacion submarina general.
