# Niveles de realismo

## Nivel 1: educativo basico

El usuario aprende nombres de mandos, instrumentos y acciones generales. No se simulan fallas complejas ni procedimientos reales completos.

Ejemplos:

- acelerar y frenar una moto;
- girar un automovil;
- usar direccion y potencia en un barco;
- interpretar altitud y velocidad en un avion pequeno.

## Nivel 2: simulacion simplificada

Se agregan variables fisicas y mecanicas basicas:

- masa;
- velocidad;
- friccion;
- potencia;
- combustible o energia;
- inercia;
- radio de giro;
- estabilidad.

## Nivel 3: simulacion tecnica

Se incluyen subsistemas y procedimientos simplificados:

- transmision;
- frenos;
- propulsion;
- navegacion;
- sensores;
- comunicaciones simuladas;
- gestion de energia;
- clima o entorno.

## Nivel 4: simulacion avanzada

Se permite mayor detalle, siempre dentro de limites seguros y publicos:

- fallas no sensibles;
- desgaste;
- modelos fisicos mas precisos;
- condiciones ambientales;
- planificacion de rutas;
- reglas operacionales.

## Nivel 5: entrenamiento certificado

Fuera del alcance del repositorio. Requiere instructores, licencias, equipos homologados, normas oficiales y supervision especializada.
