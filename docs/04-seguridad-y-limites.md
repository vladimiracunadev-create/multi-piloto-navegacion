# Seguridad y limites

## Uso previsto

Este repositorio es para documentacion, aprendizaje general, historia y diseno de simuladores. No reemplaza entrenamiento real, certificaciones, licencias ni manuales oficiales vigentes.

## Limites

No incluir:

- procedimientos tacticos reales;
- instrucciones de uso de armas;
- instrucciones para evadir controles, autoridades o sistemas de seguridad;
- informacion clasificada, restringida o no publica;
- pasos que permitan operar maquinaria peligrosa sin supervision;
- detalles tecnicos sensibles sobre sistemas militares modernos.

## Enfoque permitido

Si una maquina tiene uso militar o de alto riesgo, documentar:

- historia publica;
- funcion general;
- mandos de simulacion no sensibles;
- principios fisicos;
- roles de tripulacion a nivel general;
- reglas de seguridad;
- representacion responsable en juegos.

## Recomendacion editorial

Cuando exista duda, escribir para simulacion educativa y no para operacion real.
